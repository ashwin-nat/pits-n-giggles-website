---
title: How do I install Pits n' Giggles?
category: Getting Started
order: 1
---

Pits n' Giggles is a single standalone executable — no installer required.

1. Go to the [Releases](/releases) page and download the latest version for your platform.
2. Extract the zip file to a folder of your choice (It is recommended to create a dedicated folder for Pits n' Giggles).
3. Run the executable directly — no install steps needed.

<Callout type="tip" title="First time?">
  Follow the [Getting Started Guide](/blog/setting-up-udp-telemetry-f1) for tips on how to set up and use Pits n' Giggles for the first time.
</Callout>
