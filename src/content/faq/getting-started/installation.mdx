Pits n' Giggles is a single standalone executable — no installer required.

1. Go to the [Releases](/releases) page and download the latest version for your platform.
2. Extract the zip file to a folder of your choice.
3. Run the executable directly — no install steps needed.

<Callout type="tip" title="First time?">
  Make sure UDP telemetry is enabled in your F1 game settings before launching Pits n' Giggles. See [How do I enable UDP telemetry?](/faq#setting-up-udp-telemetry) for details.
</Callout>
