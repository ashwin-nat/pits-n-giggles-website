Pits n' Giggles supports the following EA Sports F1 games:

| Game | Status |
|------|--------|
| F1 25 | <Badge variant="supported" /> |
| F1 24 | <Badge variant="supported" /> |
| F1 23 | <Badge variant="supported" /> |
| F1 22 and older | <Badge variant="unsupported" /> |

<Callout type="info" title="UDP Format">
  Support is tied to Codemasters' UDP telemetry format. F1 23, 24, and 25 all use compatible formats that Pits n' Giggles can read.
</Callout>
